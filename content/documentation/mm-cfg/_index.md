You can configure the behavior of the Flash player using the following configuration file:
- **Windows**: `%HOMEDRIVE%\%HOMEPATH%\mm.cfg`
- **Mac**: `/Library/Application Support/Macromedia/mm.cfg`
- **Linux**: `~/mm.cfg`
